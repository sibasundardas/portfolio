Personal Portfolio Website

View Code

A fully responsive, animated, and interactive digital portfolio for showcasing projects and skills.

🔥 Project Overview

This website serves as a professional digital space where I showcase my best work — from deep learning and computer vision projects to clean, interactive front-end applications. Built with a focus on design, performance, and usability, it's optimized for both desktop and mobile devices.

🛠️ Technologies & Tools Used

  • HTML5 – For clean and semantic structure
  • CSS3 + Tailwind CSS – For modern, utility-first styling and responsiveness
  • JavaScript – For dynamic behavior and interactivity
  • Framer Motion (AOS/Animation Library) – Smooth animations and transitions
  • Git & GitHub – For version control and project hosting

💡 Features & Highlights

  1. Hero Section – A compelling intro with subtle animation that captures attention
  2. About Me – A brief overview of my background, education, and tech focus
  3. Projects – Interactive project cards with expandable details and live demo links
  4. Skills Section – A carousel/grid of technologies I’m proficient in
  5. Responsive Design – Fully optimized for mobile, tablet, and desktop
  6. Footer – Includes contact details and quick links to GitHub and LinkedIn

📌 Key Takeaways & Learning Outcomes

  • ✅ Mastered Tailwind CSS for responsive and fast UI development
  • ✅ Gained hands-on experience in modern frontend workflows
  • ✅ Focused on building clean, aesthetic, and accessible interfaces
  • ✅ Understood the importance of personal branding in tech
