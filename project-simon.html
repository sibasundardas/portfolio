Simon Game

View Code

A memory game where players repeat increasing color sequences to test focus and recall.

🎯 Project Overview

Simon Game is designed to enhance focus and memory by presenting a pattern of colors that the player must repeat. Each successful attempt adds a new color to the sequence, pushing the player to improve with every round!

🛠️ Technologies & Tools Used

  • HTML5 – For structuring the layout
  • CSS3 – For styling, transitions, and responsive visuals
  • JavaScript – Core game logic and interactivity
  • DOM Manipulation – To handle dynamic updates and user input

💡 Features & Functionality

  1. Random sequence generation for each level
  2. Visual cues for game interaction
  3. Game over detection with restart mechanism
  4. Interactive, clean UI suitable for all skill levels

📌 Key Learning Outcomes

  • ✅ Strengthened core JavaScript concepts (functions, arrays, loops, conditionals)
  • ✅ Improved DOM manipulation skills for real-time UI updates
  • ✅ Built an interactive game without using external libraries
  • ✅ Practiced responsive design and smooth user experiences

🚀 Future Enhancements

  • 🔹 Add sound effects for richer feedback
  • 🔹 Make it fully responsive on mobile devices
  • 🔹 Include high score tracking using localStorage
  • 🔹 Deploy online with improved animations and polish
