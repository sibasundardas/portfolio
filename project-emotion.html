Facial Emotion Recognition & Detection

View Code

A real-time CNN + OpenCV system to classify seven human emotions from webcam or images.

🔥 Project Overview

In this project, I built a CNN-based model that accurately classifies facial emotions from live webcam feed or images. Using OpenCV for face detection and a custom-trained CNN on the FER2013 dataset, the system identifies seven key emotions: Angry, Disgust, Fear, Happy, Neutral, Sad, and Surprise.

🛠️ Technologies & Libraries Used

  • Python – Core programming language
  • OpenCV – Real-time video stream & face detection
  • TensorFlow/Keras – Deep learning model development
  • NumPy & Pandas – Data handling and analysis
  • Matplotlib & Seaborn – Model visualization and accuracy plots
  • FER2013 Dataset – Facial expression data (35K+ images)

💡 Intuition & Approach

  1. Data Preprocessing 🧹
    Preprocessed 35,887 grayscale images (48×48) categorized into 7 emotions from FER2013. Applied normalization and reshaping for CNN compatibility.
  2. Model Architecture 🧠
    Built a Convolutional Neural Network (CNN) with multiple Conv, MaxPooling, and Dense layers to extract and classify facial features.
  3. Face Detection & Real‑Time Prediction 🎥
    Used OpenCV’s Haar Cascade Classifier to detect faces in live webcam feed and overlay predicted emotions in real-time.

📌 Key Takeaways & Learning Outcomes

  • ✅ Real-Time Emotion Recognition – Built a live system to detect and classify human emotions.
  • ✅ CNN Mastery – Gained in-depth knowledge of CNNs and model tuning.
  • ✅ Computer Vision Integration – Integrated OpenCV and deep learning seamlessly.
  • ✅ Model Evaluation – Achieved over 74% accuracy on test data with effective visualization and confusion matrix analysis.
